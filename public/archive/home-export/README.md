# Ready-to-use /home Export

This folder contains a portable, minimal version of your /home page for use in any React or Next.js project.

## Usage

1. Copy `HomePage.tsx` and `HeroSection.tsx` into your new project (e.g., into `pages/` or `app/`).
2. Ensure you have `lucide-react` and `next/image`, `next/link` installed (or replace with your own icon/image/link components).
3. Import and use `HomePage` as your main page.

## Features
- No external CSS or Tailwind required (uses inline styles for portability)
- Minimal dependencies
- Theme switcher included (light/dark toggle)
- All visuals and layout from your original /home page
